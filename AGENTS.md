# 项目说明 · AGENTS.md

全栈 AI 驱动的现代管理系统。后端 Gin 存放于 `backend/`，前端 React 存放于 `frontend/`，前后端分离。

## 布局偏好（默认采用，除非另有说明）
- 主题：**如意国风科技蓝**（深海军蓝底 + 科技蓝/青色高亮 + 国风金线点缀，如意云纹徽标）。
- 整体：**满屏**布局，宽高 = 屏幕宽高（100vw / 100vh），不出现整体页滚动。
- **顶部栏**：固定高度（约 60px），横向铺满。
- **左侧栏**：固定宽度（约 224px），纵向铺满主体高度，内部可滚动。
- **内容区**：自适应剩余空间，高度超出时**仅内容区内部滚动**，不影响顶栏与侧栏。
- 若后续需求与上述布局不一致，先向用户确认，不要直接改动默认布局。

## 前端约定
- 框架：React + Vite。
- 接口：当前采用 `src/mock/data.js` 的 mock 实现，后续替换为真实 Gin 接口。
